Используя api от приватбанка https://api.privatbank.ua/#p24/atm 
написать приложение, которе выводит список банкоматов в RecyclerView 
для заданного города в Украине (по умолчанию - в Днепре).

Использовать Gson для сериализации, OkHttp для запроса данных.

В списке отобразить информацию: 
1. Название места, где стоит банкомат
2. Есть ли доступ к банкомату в текущее время суток

При клике по элементу списка отобразить детальную
информацию во фрагменте:
1. Адрес места, где стоит банкомат
2. Координата местоположения (широта и долгота)
3. График работы
